# Ecommerce-data-analysis-project
End to end data analysis project using Python, SQL and Power BI
E-commerce Data Analysis Project
# Overview

  In this project, I worked on an e-commerce dataset to analyze customer behavior, sales trends, and overall business performance using SQL, Python, and Power BI.

# Tools Used
  SQL
  Python (Pandas, Matplotlib)
  Power BI
# What I Did
1. SQL
  Created database and imported data
  Found total revenue, orders, and customers
  Analyzed monthly revenue and top customers
2. Python
  Loaded and cleaned data
  Checked missing values
  Performed EDA
  Created charts for monthly trends
  RFM Analysis
  Segmented customers into:
  High Value
  Regular
  At Risk
3. Power BI
  Built dashboard for:
  Revenue trends
  Customer insights
  Sales performance
# Key Insights
  Many customers are at risk
  High-value customers generate most revenue
  Sales vary month to month
# Conclusion
  This project helped me learn how to analyze real data and find useful insights using different tools.
